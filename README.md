## 安装

```bash
pip install two

# 或者
git clone https://github.com/Ailln/two.git
cd two
python setup.py install
```

## 使用

```python
>>> import two
>>> two.two()
错的不是我，是世界！【叛逆的鲁鲁修R2】
>>> two.two()
你还差的远呢！【网球王子】
```
